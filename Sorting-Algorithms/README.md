# Overview:
A small interactive console program that lets you type 9 numbers, pick a classic sorting algorithm, and see the result plus the execution time in microseconds.

# Implemented algorithms:
  Insertion Sort - O(n²) 
  Selection Sort - O(n²) 
  Heap Sort - O(n log n) 
  Merge Sort - O(n log n) 
  Quick Sort - O(n²) (average O(n log n)) 
# Quick Start:
  Compile
g++ main.cpp sorting.cpp sorting.h -o sorting_demo
  Run
./sorting_demo OR ./sorting_demo.exe
