A small interactive console program that lets you type 9 numbers, pick a classic sorting algorithm, and see the result plus the execution time in microseconds.

Implemented algorithms:

# Algorithm        | Time Complexity (worst)
1.) Insertion Sort  O(n²) |
2.) Selection Sort  O(n²) |
3.) Heap Sort       O(n log n) |
4.) Merge Sort      O(n log n) |
5.) Quick Sort      O(n²) (average O(n log n)) |

- Quick Start:

- 1. Compile
g++ main.cpp sorting.cpp sorting.h -o sorting_demo

- 2. Run
./sorting_demo || ./sorting_demo.exe
